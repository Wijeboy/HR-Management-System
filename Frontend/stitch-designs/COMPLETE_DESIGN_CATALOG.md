### HTML File Characteristics
- **Format:** Standard HTML with inline CSS/JavaScript

- **Styling:** Mix of inline styles and CSS classes
- **Images:** Some use Google User Content URLs
- **Responsiveness:** Primarily desktop-focused designs

### Empty HTML Files - Root Causes
Stitch API returns 0-byte HTML downloads when:
1. Screen was created without HTML code generation
2. Code generation failed during Stitch processing
3. Screen uses features not supported by HTML export
4. Project regeneration didn't include HTML for all screens

### File Naming Convention
```
[number]-[descriptive-name].[extension]
```
- Numbers: 01-23 (sequential)
- Names: Kebab-case, descriptive
- Extensions: .png (screenshots), .html (code)
